
	Utilizing the research system:
	
- Create a new .config file

- Create a patch for /zb/researchTree/data.config that has only  this line:
	[{"op" : "add", "path" : "/researchFiles/", "value" : "MYFILEPATH"}]
	
- Replace MYFILEPATH with the path to your previously created .config file

- Fill out your .config file following the template named "research_template.config" provided in the same folder as this document
